Settings & privacy

Tracked sites

TypeLens only counts on the sites listed here. Toggle any off, or add your own. Adding a site asks Chrome for permission to run on it.

Active hours

Optionally only count during set hours. If your window ends while you're typing, TypeLens gently asks before stopping — it never cuts you off silently.

Pause

Pause counting anytime with the keyboard shortcut Ctrl+Shift+U, from the popup, or here.

Your data

Everything lives on this device. Export a copy or erase it all — your call, instantly, no questions asked.

What happens to every keystroke

This is the whole story, in plain language. The code backs up every sentence — it's open source, read it.

  1. You press a key. A listener notices a key went down inside a text field.
  2. It asks one yes/no question: "was that a single printable character?" It checks the length of the key — never which key it was.
  3. If yes, a number goes up by one. That number is the only thing that exists. The keystroke itself is already gone.
  4. Every couple of seconds, that number is added to a daily total for the site you're on, saved in your browser's local storage.
  5. That's it. Forever. The data model is: dates → site names → counts. There is nowhere for your words to live, because nothing ever holds them.
No text is ever stored, anywhere.
No network requests. No servers. No accounts. No sync.
Works fully offline, identically, forever.
